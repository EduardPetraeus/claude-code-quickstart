## Broad Thinking Templates

Use these BEFORE building anything. They force divergent thinking so you don't lock into the first idea.

| Template | What It Does | Best For |
|---|---|---|
| [strategy-diverge.md](broad-thinking/strategy-diverge.md) | Generate 5 fundamentally different strategies | New projects, major decisions |
| [metaphor-reframe.md](broad-thinking/metaphor-reframe.md) | Map your problem onto other domains | Architecture, debugging |

## How to Use These

1. Pick the template that matches your situation
2. Copy the prompt template from the file
3. Fill in the `[BRACKETS]` with your specific context
4. Paste into Claude Code (or any AI chat)

The prompt skeleton is the foundation — learn it first, then add the others when you need them.
