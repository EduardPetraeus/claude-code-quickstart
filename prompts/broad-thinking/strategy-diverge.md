# Strategy Diverge

## Purpose
Force divergent strategic thinking by generating fundamentally different approaches.

## Prompt Template

```
I need you to think broadly about this problem: [PROBLEM]

Generate 5 fundamentally different strategies. Not variations of the same idea — truly different approaches that a different type of person would suggest:

1. The engineer's approach (systematic, scalable)
2. The entrepreneur's approach (fast, scrappy, revenue-first)
3. The artist's approach (beautiful, elegant, unconventional)
4. The scientist's approach (hypothesis-driven, data-first)
5. The contrarian's approach (what everyone else is wrong about)

For each strategy:
- Core idea in one sentence
- Why it might be the best approach
- What you'd have to believe for this to work
- Biggest risk

Then: which strategy would you bet your own money on, and why?
```

## When to Use
- Start of any new project or major feature
- When feeling stuck on a single approach
- Before committing to an architecture or business model
